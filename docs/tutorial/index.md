# Tutorial

A practical guide to building workflows in Graflow — from your first task to advanced patterns.

This guide teaches you how to define tasks and build workflows through practical examples.

8. [Complex Workflows](/docs/tutorial/core-concepts/complex-workflows) — Diamond patterns and multi-instance

9. [Dynamic Task Generation](/docs/tutorial/advanced/dynamic-tasks) — Runtime task addition and control flow
