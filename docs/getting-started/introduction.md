# Introduction

Graflow is a workflow engine designed to be **reliable**, **explainable**, and **scalable**.

## What is Graflow?

Graflow allows you to define complex data pipelines and workflows using simple YAML configuration. It handles task dependencies, parallel execution, error recovery, and provides full visibility into every step of your workflow execution.

## Key Features

- **Reliable**: Built-in fault tolerance and automatic recovery ensure your workflows complete successfully
- **Explainable**: Detailed logs, lineage tracking, and clear error messages make debugging straightforward
- **Scalable**: Seamlessly scale from local development to production clusters

## When to use Graflow

Graflow is ideal for:

- Data pipelines and ETL workflows
- Machine learning training pipelines
- Batch processing jobs
- Any complex task orchestration
